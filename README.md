## Descripción del proyecto
Sistema de información web para una veterinaria desarrollado con Blazor Server .NET 8.
Integra datos de la API del profesor con una base de datos propia en Azure SQL.

## Tecnologías utilizadas
- Blazor Server (.NET 8)
- Entity Framework Core 8
- Azure SQL Database
- HttpClient con GetFromJsonAsync

## API consumida

- **Endpoint 2:** /api/salud/servicios/especialidad/4
